# 📘 HTML + CSS

---

# 1. Що таке HTML і CSS

**HTML (HyperText Markup Language)** — мова розмітки, яка визначає структуру сторінки.
👉 Тобто: які елементи є на сторінці (текст, кнопки, картинки).

**CSS (Cascading Style Sheets)** — мова стилів, яка визначає вигляд сторінки.
👉 Тобто: як ці елементи виглядають (кольори, відступи, розміри).

📌 Аналогія:

* HTML = каркас будинку
* CSS = ремонт і дизайн

---

# 2. Як працює браузер

1. Завантажує HTML
2. Перетворює його в DOM (структура як дерево)
3. Завантажує CSS
4. Застосовує стилі до елементів
5. Малює сторінку

👉 Важливо: браузер НЕ просто читає код — він будує структуру і тільки потім її відображає

---

# 3. Базова структура HTML

```html
<!DOCTYPE html> <!-- говорить браузеру що це HTML5 -->
<html> <!-- корінь документа -->
<head> <!-- службова інформація -->
    <title>Сторінка</title> <!-- назва вкладки -->
</head>
<body> <!-- весь видимий контент -->
</body>
</html>
```

---

# 4. Теги і атрибути

```html
<a href="https://example.com">Link</a>
```

👉 `a` — тег посилання
👉 `href` — атрибут, який задає адресу переходу

```html
<img src="img.png" alt="кіт">
```

👉 `img` — зображення
👉 `src` — шлях до картинки
👉 `alt` — альтернативний текст

📌 alt показується якщо:

* картинка не завантажилась
* користувач використовує screen reader

---

# 5. Вкладеність

```html
<div>
    <p><span>Текст</span></p>
</div>
```

👉 span знаходиться всередині p
👉 p знаходиться всередині div

📌 правило: теги повинні правильно відкриватись і закриватись

---

# 6. Основні HTML теги

## Текст

* `h1-h6` — заголовки (h1 найважливіший)
* `p` — параграф тексту
* `span` — маленький контейнер для тексту

## Контейнери

* `div` — універсальний блок без сенсу
* `section` — логічний розділ сторінки
* `article` — незалежний контент (пост, новина)

## Медіа

* `img` — картинка
* `video` — відео

## Списки

* `ul` — маркований список
* `ol` — нумерований список
* `li` — елемент списку

---

# 7. Семантика HTML

## Структура сторінки

```html
<body>
    <header></header> <!-- верх сторінки -->
    <main></main> <!-- основний контент -->
    <footer></footer> <!-- низ сторінки -->
</body>
```

---

## main

👉 головний контент сторінки
👉 тільки 1 на сторінці

## section

👉 логічний блок
👉 майже завжди має заголовок

## article

👉 самостійний контент
👉 може існувати окремо

## div

👉 просто контейнер для стилів

---

## Як мислити

* сторінка → main
* блок → section
* контент → article
* стилі → div

---

# 8. CSS

## Як підключається

```html
<link rel="stylesheet" href="styles.css">
```

👉 підключає зовнішній файл стилів

---

# 9. Основні стилі

```css
color: red; /* колір тексту */
background: black; /* фон */
margin: 10px; /* зовнішній відступ */
padding: 10px; /* внутрішній відступ */
border: 1px solid black; /* рамка */
width: 100px; /* ширина */
height: 100px; /* висота */
```

---

# 10. display

```css
display: block; /* займає всю ширину */
display: inline; /* тільки по контенту */
display: inline-block; /* можна задавати розміри */
display: none; /* приховує елемент */
```

---

# 11. Flexbox

```css
.container {
    display: flex;
}
```

👉 робить елементи гнучкими

```css
flex-direction: row;
justify-content: center; /* вирівнювання по горизонталі */
align-items: center; /* вирівнювання по вертикалі */
gap: 10px; /* відступи між елементами */
```

---

# 12. Hover і взаємодія

```css
button:hover {
    background: red; /* змінюється при наведенні */
}

button:active {
    transform: scale(0.95); /* ефект натискання */
}

button:focus {
    outline: 2px solid blue; /* фокус (наприклад через Tab) */
}
```

---

# 13. Часто використовувані стилі

```css
border-radius: 10px; /* закруглення */
box-shadow: 0 0 10px black; /* тінь */
cursor: pointer; /* курсор рука */
overflow: hidden; /* приховує зайве */
```

# 🔥 ДОДАТКОВО

# 14. CSS Box Model

Кожен елемент складається з:

* content — сам контент
* padding — внутрішній відступ
* border — рамка
* margin — зовнішній відступ

👉 це критично для розуміння layout

---

# 15. Position

```css
position: static; /* за замовчуванням */
position: relative; /* відносно себе */
position: absolute; /* відносно батька */
position: fixed; /* фіксовано на екрані */
position: sticky; /* прилипає при скролі */
```

---

# 16. Flex vs Grid

* Flex — для одного напрямку (ряд або колонка)
* Grid — для двох напрямків (таблиця)

---

# 17. Адаптивність

```css
@media (max-width: 768px) {
    body {
        background: red;
    }
}
```

👉 стилі змінюються залежно від розміру екрану

---

# 18. Практичні поради

* не використовуй div всюди
* додавай alt до зображень
* не пиши inline стилі
* давай зрозумілі назви класам

---

# 19. Типові помилки

* неправильна вкладеність
* відсутність alt
* копіпаст стилів
